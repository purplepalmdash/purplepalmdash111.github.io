GitLab仓库

在前面搭建的GitLab里创建一个新仓库，用于存储Packer.io脚本。


在编译机器的仓库里，运行以下命令，添加自己到新创建的仓库里:

$ cd existing_folder
$ git init
$ git remote add origin http://192.168.1.79:10080/root/BuildUbuntu.git
$ git add .
$ git commit
$ git push -u origin master

提交完毕之后，在GitLab服务器上就可以看到新添加的代码了。
