Steps

Reconfigure the LC_ALL, etc:

$ sudo vim /etc/environment 
LC_ALL=en_US.UTF-8
LANG=en_US.UTF-8
$ sudo locale-gen "en_US.UTF-8"
$ sudo dpkg-reconfigure locales
$ sudo reboot

Be Sure to use latest repository, like aliyun.com.

$ sudo apt-get update -y
$ sudo bash
