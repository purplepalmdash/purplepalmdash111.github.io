1\. 安装python-dev, 准备环境
```
$ apt-get install python-dev
$ wget https://pypi.python.org/packages/source/R/RPi.GPIO/RPi.GPIO-0.5.3a.tar.gz
$ tar xzvf  RPi.GPIO-0.5.3a.tar.gz
$ python setup.py install
```
