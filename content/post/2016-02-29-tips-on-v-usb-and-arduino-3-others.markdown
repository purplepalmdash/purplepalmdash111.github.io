### custom_class延伸
连线LED 11，修改代码后即可控制11 GPIO口的LED亮灭

custom_class_demo.pde:    

```
const int ledPin = 11;
```

custom_class.h:    

```
class custom_classDevice {
 public:
  custom_classDevice() {
    usbInit();
    ledPin = 11;        // pin 11 as default ledPin
  }
```

而后连接一个220欧电阻+LED到ARDUINO的PIN 11上，即可通过USB控制LED亮灭。
